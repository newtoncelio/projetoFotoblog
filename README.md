## 💻 Projeto  
Esse é um projeto Web de uma blog para exibir uma galeria de imagens.

## 🚀 Tecnologias
Esse projeto foi desenvolvido durante o curso da Rocketseat com as seguintes tecnologias:

- HTML
- CSS (animation, transition, grid)
- APIs
- Git e Github
